# alphiqa
ALPHIQA COMPILER
- Alphiqa is a comiler that compiles .aqai files down to C3 files. It was first used on day 7 of development in basic print statements.
The tkinter GUI runs on python3.
It is designed to be a midleground between Python (high level: slow runtime, easy syntax) and C3 (lower level: fast runtime, hard syntax). Alphiqa is hybrid.
